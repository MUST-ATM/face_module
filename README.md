# Face-module

## Description
This module is part of the project `MUST-ATM` and is responsible for the face detection and recognition. It uses the `dlib` library to detect faces and the `OpenCV` library to recognize them.

## Requirements
- Python >=3.6
- OpenCV >=3.4.2
- Numpy >=1.16.2
- Dlib >=19.16.0
